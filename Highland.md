[Highland 2][highland] users can preview Fountain or Markdown drafts in Marked by **dragging the Highland document** from Finder (or from the proxy icon in the window title bar) onto Marked's icon or into an open Marked window.

Marked updates the preview when the file changes on disk, just like a normal Markdown file.
