[Hookmark](https://hookproductivity.com/) links can point to files and assets that your Markdown references, including images.

## Hookmark URL resolution

Marked can resolve `hook://` links for images and other assets in a file when Hookmark integration is enabled.

## Enable in Settings

Open {% prefspane Apps %} and turn on the Hookmark-related option. Once enabled, Marked resolves Hookmark URLs while rendering so linked assets appear normally in preview and export.
