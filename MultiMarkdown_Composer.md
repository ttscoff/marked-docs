[MultiMarkdown Composer][mmc] treats Marked as an external preview engine.

Open Composer's **Preview** preferences and choose Marked as the preview application, or press {% kbd opt cmd P %} **Open in Marked** from the active document whenever you want Marked's style system, export tools, or proofing overlays on the same Markdown source Composer is editing.

Marked watches the Composer file path (or exchanged content, depending on version) and refreshes whenever Composer saves.

[mmc]: https://multimarkdown.com/composer4/
