Zoom Overview shrinks the preview text so you can see more of the document at once&#8212;useful for spotting structure, scanning for a section, or judging length without opening the [Mini Map](Document_Navigation.html#minimap).

## Using Zoom Overview

Press {% kbd z %} in the preview to zoom out. Press {% kbd z %} again to return to normal size. While zoomed, you can also press **Escape** or use the dismiss control on the {% kbd z %} tip to zoom back in.

## Word Repetition visualization

The same {% kbd z %} shortcut applies when **Word Repetition** ({% kbd ctrl cmd w %}) is open: toggling the zoom helps you see where repeated words cluster in the document.
