## Do not translate (keep English)

- File names and code identifiers (`Exporting.md`, `Exporting.html`)
- `{% prefspane General %}`, `{% kbd cmd S %}`, `{{cmd}}`, `{{gear}}`, etc.
- URL schemes: `x-marked-3://`
- Version numbers and build metadata
