# <%= @title %>

[Hookmark](https://hookproductivity.com/)-Links können auf Dateien und Assets verweisen, die Ihr Markdown nutzt, einschließlich Bildern.

## Hookmark-URL-Auflösung

Marked kann `hook://`-Links für Bilder und andere Assets in einer Datei auflösen, wenn die Hookmark-Integration aktiviert ist.

## In den Einstellungen aktivieren

Öffnen Sie {% prefspane Apps %} und aktivieren Sie die Hookmark-bezogene Option. Sobald aktiviert, löst Marked beim Rendern Hookmark-URLs auf, sodass verknüpfte Assets in der Vorschau und im Export normal angezeigt werden.
