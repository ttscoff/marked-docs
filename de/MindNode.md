# <%= @title %>

Ziehen Sie ein [MindNode][mn]-Dokument auf Marked (oder verwenden Sie je nach MindNode-Version den MindNode-Befehl **In Marked öffnen** / Vorschau öffnen). Marked behandelt die Datei wie einen Markdown-Export von MindNode, sodass die Hierarchie Ihrer Map in der Vorschau zu Überschriften und Listen wird.

## Streaming-Vorschau (MindNode 2026.3+)

I> **MindNode 2026.3** und neuer fügen Marked eine [Streaming-Vorschau](Streaming_Preview.html) hinzu. Wählen Sie in MindNode **Ansicht → Vorschau in Marked**, um sie zu öffnen; die Vorschau wird live aktualisiert, wenn Sie die Map ändern.

## MindNode Classic im Vergleich zu MindNode 7+

W> Die aktuelle Integration ist auf **MindNode Classic**-Bundles ausgelegt. Neuere MindNode-Versionen verwenden ein anderes Paketformat. Wenn das Öffnen einer Map fehlschlägt oder Marked ein **„Please open from MindNode“**-Dialogblatt anzeigt, bereiten Sie die Map über das MindNode-Menü **Advanced** vor (häufig **Open in another application** / Marked) oder exportieren Sie ein kompatibles Bundle, statt ein rohes `.mindnode`-Paket zu ziehen, das nicht für die Vorschau vorbereitet wurde.

Da die Datei von MindNode vorbereitet wird, bevor Marked sie liest, speichern Sie sie immer in MindNode, bevor Sie mit einer Aktualisierung in Marked rechnen.
