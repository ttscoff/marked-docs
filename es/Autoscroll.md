# <%= @title %>

El desplazamiento automático mueve lentamente la vista previa para lectura o revisión con manos libres. Está separado de la [Tabla de contenido](Document_Navigation.html) y el [Minimapa](Document_Navigation.html#minimap); desplaza la vista actual automáticamente.

## Iniciar y detener

Active el desplazamiento automático presionando {% kbd s %}. La vista previa avanza a la velocidad actual.

Un indicador en la parte inferior izquierda muestra la velocidad. Ajústelo con las teclas de flecha **Arriba** y **Abajo**; Mantenga presionada la tecla **Shift** con las flechas para pasos más grandes.

Retorno ({% kbd return %}) pausa o reanuda el desplazamiento.

{% kbd shift s %} invierte la dirección de desplazamiento mientras el desplazamiento automático está activo.

Presione {% kbd s %} nuevamente o Escape para salir del modo de desplazamiento automático.

Cuando utilice Marked como un teleprompter, puede agregar pausas programadas en la fuente de Markdown; consulte [Pausas de desplazamiento automático](Special_Syntax.html#pauses) en [Sintaxis especial de Marked](Special_Syntax.html).
