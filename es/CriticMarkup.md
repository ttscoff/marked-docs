[CriticMarkup](http://criticmarkup.com/) permite a los editores proporcionar comentarios dentro de los documentos de Markdown. Cuando se habilita en {% prefspane  Proofing %}, la vista previa marcada proporcionará tres pestañas cada vez que se detecte la sintaxis de CriticMarkup: la versión de marcado que muestra inserciones, eliminaciones y comentarios en línea, la versión original antes de las ediciones y la versión editada tal como aparecería si se aceptaran todas las ediciones.

Puede alternar la compatibilidad con CriticMarkup para cualquier documento desde el menú de ajustes o usando {% kbd ctrl shift C %}.

Cambie de vista rápidamente en una vista previa de CriticMarkup con {% kbd shift cmd [ %} y {% kbd shift cmd ] %}

Cuando CriticMarkup está activo, el recuento de palabras y las estadísticas del documento cambiarán según la vista en la que se encuentre actualmente. Esto permite comparar las estadísticas básicas y el recuento de palabras, oraciones y caracteres entre las versiones original y editada de un documento.

Para obtener más información sobre CriticMarkup y la integración con sus editores de texto favoritos, consulte la [página de inicio de CriticMarkup](http://criticmarkup.com/).
