## Deshabilitar ID de encabezado automático

Marked incluye una opción para deshabilitar la generación automática de ID de encabezado. Puedes encontrar esta opción en el {% prefspane Processor %}.

## ID de notas al pie aleatorias

En el panel **Procesador**, puede marcar "Usar ID de notas al pie aleatorias" para generar identificadores de notas al pie aleatorias, lo que ayuda a evitar conflictos cuando se muestran varios documentos en una página web. Esta opción sólo está disponible cuando se utiliza el procesador MultiMarkdown.

## Procesar Markdown dentro de HTML

De forma predeterminada, los procesadores de Markdown suelen ignorar la sintaxis de Markdown dentro de las etiquetas de bloque HTML. Esta opción obliga a Marked a continuar procesando dentro de los elementos del bloque. Tenga en cuenta que algunas marcas pueden causar problemas.
