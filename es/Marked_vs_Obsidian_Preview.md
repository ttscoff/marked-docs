# <%= @title %>

Obsidian incluye una vista previa de Markdown incorporada. Marked 3 es una aplicación de publicación y vista previa dedicada que funciona con Obsidian y muchos otros editores. Esta página le ayuda a decidir cuándo cada herramienta es la adecuada.

## Cuando la vista previa incorporada de Obsidian es suficiente

*Solo escribes dentro de Obsidiana
* La vista previa básica y los complementos satisfacen sus necesidades
* No necesita flujos de trabajo de exportación o revisión independientes
* Quieres todo en una sola aplicación con una configuración mínima

Obsidian (y aplicaciones como **Typora**) son ideales cuando todo lo que necesitas es una solución simple de una sola aplicación.

## Cuando Marcado es la mejor opción

* Quieres una vista previa en vivo de **cualquier** editor, no solo de Obsidian
* Necesita exportación profesional (PDF, DOCX, EPUB) con CSS personalizado
* Quiere revisión avanzada, estadísticas de documentos y resaltado de palabras clave
* Trabajas con Scrivener, Word o MarsEdit junto con Markdown
* Quieres temas con calidad de publicación a través de [Estilos personalizados](Custom_Styles.html)

La vista previa de Marked incluye muchas funciones que las vistas previas integradas en Obsidian, Typora y aplicaciones similares no ofrecen por sí solas, o que requieren complementos y soluciones alternativas para aproximarse.

## Comparación lado a lado

| Característica | Marcado 3 | Vista previa de obsidiana |
| :-- | :-- | :-- |
| Vista previa en vivo desde cualquier editor de Mac | Sí | Sólo limas de obsidiana |
| Vista previa dedicada + aplicación de exportación | Sí | Aplicación de notas |
| Temas CSS personalizados | Sí | Temas/complementos |
| Herramientas de revisión | Avanzado | Básico |
| Exportación de PDF, DOCX y EPUB | Sí | Complementos |

Para obtener una comparación más amplia que incluya Typora y Marked 2, consulte la [página de comparación de marketing](https://markedapp.com/markdown-preview/).

## Utilice ambos: Obsidiana + Flujo de trabajo marcado

No es necesario reemplazar la vista previa de Obsidian (ni renunciar a la edición de ventana única de Typora) para beneficiarse de Marked. Muchos escritores mantienen su editor preferido y agregan Marcado para funciones de vista previa y opciones de exportación que la otra aplicación no ofrece.

1. Escriba en Obsidian (o Typora, VS Code, etc.) como de costumbre.
2. Arrastre la carpeta de su bóveda a Marked o use el [complemento Marked 3 Obsidian](Obsidian.html).
3. Marked observa la bóveda y obtiene una vista previa de la nota editada más recientemente.
4. Utilice Marcado para revisión, estadísticas de documentos, temas personalizados y exportación cuando el borrador esté listo. Consulte [Exportar](Exporting.html).
