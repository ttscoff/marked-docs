# <%= @title %>

[MultiMarkdown Composer][mmc] trata a Marked como un motor de vista previa externo.

Abra las preferencias de **Vista previa** de Composer y elija Marked como aplicación de vista previa, o presione {% kbd opt cmd P %} **Abrir en Marked** desde el documento activo cuando desee que el sistema de estilo, las herramientas de exportación o las superposiciones de revisión de Marked estén editando en la misma fuente de Markdown que Composer está editando.

Marked observa la ruta del archivo de Composer (o el contenido intercambiado, según la versión) y se actualiza cada vez que Composer guarda.

[mmc]: https://multimarkdown.com/composer4/
