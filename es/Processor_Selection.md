# <%= @title %>

## Selección de procesador

[MultiMarkdown](http://fletcherpenney.net/multimarkdown/ "MultiMarkdown") (de Fletcher Penney) es una extensión de [Markdown](http://daringfireball.net/projects/markdown/ "Daring Fireball: Markdown") (de John Gruber). Se respeta todo el formato estándar de Markdown y MultiMarkdown proporciona sintaxis adicionales para elementos como tablas y notas al pie.

## Sintaxis

Hay una guía de sintaxis de Markdown en el menú de ayuda, o puede consultar la [documentación original de John Gruber](http://daringfireball.net/projects/markdown/syntax).

Fletcher Penney documenta las extensiones MultiMarkdown en [este documento](https://github.com/fletcher/MultiMarkdown/blob/master/Documentation/MultiMarkdown%20User%27s%20Guide.md).

## Representación alternativa

Puedes elegir entre MultiMarkdown, Kramdown, CommonaMark y Discount en el {% prefspane Processor %}. Cada uno ofrece funciones adicionales con una sintaxis ligeramente diferente (consulte {% appmenu Help, Markdown Reference %} para obtener más detalles). En la mayoría de los casos, se prefiere MultiMarkdown, pero si tiene archivos que ya utilizan una sintaxis específica del procesador o tiene problemas para procesar un archivo Markdown con MultiMarkdown, seleccione Commonmark como su procesador predeterminado.
