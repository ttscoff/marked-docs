Abra siempre documentos nuevos en modo de revisión
: Los documentos nuevos mostrarán de forma predeterminada palabras clave destacadas.

Habilitar CriticMarkup automáticamente
: Habilite el procesamiento [CriticMarkup](CriticMarkup.html) automáticamente cuando se detecte la sintaxis adecuada.

Resaltar errores de sintaxis de Markdown
: resalte los enlaces de Markdown rotos y otras marcas que puedan haberse convertido incorrectamente.

Mostrar barra lateral de comentarios de forma predeterminada
: abre la barra lateral de comentarios cuando se carga un documento con comentarios o anotaciones.

Incluir notas a pie de página en la barra lateral de comentarios
: cuando la barra lateral de comentarios esté abierta, incluya notas al pie junto a los comentarios. Cuando está deshabilitado, solo aparecen comentarios (CriticMarkup, Fountain, Word y similares) en la barra lateral.

Velocidad de lectura (ppm)
: Este número (palabras por minuto) se utiliza para calcular el tiempo de lectura en pantallas de estadísticas de texto. La velocidad media de lectura para adultos es de 200 a 300 palabras por minuto.

### Palabras clave

[Definir listas de palabras clave a resaltar](Keyword_Highlighting.html). Utilice las pestañas para editar cada lista:

evitar
: Palabras y frases (una por línea) para marcar como usadas en exceso o que vale la pena revisar. Las coincidencias no distinguen entre mayúsculas y minúsculas y deben ser palabras o frases completas.

Usar alternativo
: Sugiera una redacción alternativa para las frases enumeradas.

Resaltar
: resalte palabras o frases arbitrarias para enfatizarlas durante la revisión.

Ignorar repeticiones
: Palabras que se excluirán del análisis **Visualizar repetición de palabras** (una por línea).

Restaurar frases predeterminadas
: Restablece la lista de palabras clave actual a los valores predeterminados de Marked (por ejemplo, la lista de evitar [Campaña en inglés sencillo](http://www.plainenglish.co.uk)).
