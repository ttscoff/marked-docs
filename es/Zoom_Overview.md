# <%= @title %>

Vista general de zoom reduce el texto de vista previa para que pueda ver más del documento a la vez, lo que resulta útil para detectar la estructura, escanear una sección o juzgar la longitud sin abrir el [Mini mapa](Document_Navigation.html#minimap).

## Uso de la descripción general de Zoom

Presione {% kbd z %} en la vista previa para alejar la imagen. Presione {% kbd z %} nuevamente para volver al tamaño normal. Mientras hace zoom, también puede presionar **Escape** o usar el control de descartar en la punta {% kbd z %} para volver a acercar.

## Visualización de repetición de palabras

El mismo atajo {% kbd z %} se aplica cuando **Repetición de palabras** ({% kbd ctrl cmd w %}) está abierto: alternar el zoom le ayuda a ver dónde se agrupan las palabras repetidas en el documento.
