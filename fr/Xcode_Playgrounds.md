# <%= @title %>

Ouvrez un paquet `.playground` Swift dans Marked. Marked lit `contents.swift` (ainsi que les pages associées dans les playgrounds multi-pages) et sépare les **commentaires Markdown** du code Swift exécutable.

- Les commentaires Markdown s'affichent avec le style de votre choix.
- Le code Swift exécutable est affiché dans des blocs de code délimités avec coloration syntaxique, adaptés à la lecture ou à l'impression.

C'est idéal pour les extraits de programmation lettrée, les exemples pédagogiques, ou pour documenter des expérimentations basées sur des playgrounds sans avoir à exporter au préalable vers un fichier Markdown séparé.
