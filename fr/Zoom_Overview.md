# <%= @title %>

Vue d'ensemble par zoom réduit le texte de l'aperçu afin de voir davantage du document en une seule fois : utile pour repérer la structure, parcourir une section, ou évaluer la longueur sans ouvrir la [mini-carte](Document_Navigation.html#minimap).

## Utiliser Vue d'ensemble par zoom

Appuyez sur {% kbd z %} dans l'aperçu pour effectuer un zoom arrière. Appuyez à nouveau sur {% kbd z %} pour revenir à la taille normale. Pendant le zoom, vous pouvez également appuyer sur **Échap** ou utiliser le contrôle de fermeture sur l'infobulle {% kbd z %} pour effectuer un zoom avant.

## Visualisation des répétitions de mots

Le même raccourci {% kbd z %} s'applique lorsque **Répétitions de mots** ({% kbd ctrl cmd w %}) est ouvert : activer/désactiver le zoom aide à repérer où les mots répétés se regroupent dans le document.
