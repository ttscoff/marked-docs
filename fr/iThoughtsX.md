Les cartes mentales [iThoughtsX][it] enregistrées sous forme de packages `.itmz` peuvent être prévisualisées directement dans Marked une fois que **Markdown Preview** est activé dans iThoughtsX.

## Activer le canal d'exportation Markdown

Dans les préférences d'iThoughtsX, sous **Avancé**, définissez **Aperçu Markdown** (le libellé peut indiquer « Markdown » ou « Aperçu Markdown » selon la version), puis choisissez si les sujets exportés incluent :

- **Tous les sujets** : étendue complète de la carte, ou
- **Sujets visibles** : uniquement les branches que vous avez développées dans le canevas.

Marked lit la représentation Markdown à partir de la carte packagée et la restitue à l'aide de votre feuille de style active.

Utilisez les commandes **Mode Plan** dans le menu {% appmenu Gear %} de Marked lorsque vous souhaitez une numérotation de plan de style APA ou décimal sur le plan converti.

Enregistrez dans iThoughtsX pour envoyer les mises à jour dans Marked ; l'observateur remarque automatiquement les changements du système de fichiers.
