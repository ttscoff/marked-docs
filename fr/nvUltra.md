# <%= @title %>

[nvUltra][nvu] indexe les blocs-notes en texte brut sur le disque et peut transmettre la ou les notes sélectionnées à Marked avec un raccourci intégré.

## Prévisualiser le(s) fichier(s) dans Marked

Dans le **Navigateur de dossiers** de nvUltra (ou partout où vous parcourez les notes), **contrôlez-cliquez** sur le(s) fichier(s) pour ouvrir le menu contextuel et choisissez **Aperçu du(des) fichier(s) dans Marked**.

![][nvultramenu]

Marked ouvre les aperçus normaux sauvegardés sur des fichiers afin que votre MultiMarkdown s'affiche avec les styles Marked, les signets, les bascules de mini-carte et les options d'exportation. Utilisez-le lorsque vous souhaitez passer directement de nvUltra sans faire glisser de fichiers vers le Dock.

## Alternative : surveillance des dossiers

Besoin de Marked pour suivre **la note que vous avez touchée le plus récemment** dans un dossier entier ? Déposez plutôt ce dossier de bloc-notes sur Marked ; voir **[Surveillance de dossiers](Folder_Watching.html)** pour la surveillance de style nvALT, le défilement jusqu'aux modifications et en quoi ce flux de travail diffère de l'ouverture de fichiers explicites.
