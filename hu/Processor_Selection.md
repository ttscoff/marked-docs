# <%= @title %>

## Processzor kiválasztása

A [MultiMarkdown](http://fletcherpenney.net/multimarkdown/ "MultiMarkdown") (Fletcher Penney) a [Markdown](http://daringfireball.net/projects/markdown/ "Daring Fireball: Markdown") kiterjesztése (Jon Gruber). Az összes szabványos Markdown formázást figyelembe veszi, és a MultiMarkdown további szintaxisokat biztosít az olyan elemekhez, mint a táblázatok és lábjegyzetek.

## Szintaxis

A súgó menü alatt található egy Markdown szintaxis útmutató, vagy tekintse meg [John Gruber eredeti dokumentációját](http://daringfireball.net/projects/markdown/syntax).

A MultiMarkdown kiterjesztéseket Fletcher Penney dokumentálja [ebben a dokumentumban](https://github.com/fletcher/MultiMarkdown/blob/master/Documentation/MultiMarkdown%20User%27s%20Guide.md).

## Alternatív renderelés

A {% prefspane Processor %} alatt választhat a MultiMarkdown, Kramdown, CommonaMark és Discount között. Mindegyik további szolgáltatásokat kínál némileg eltérő szintaxissal (további részletekért lásd: {% appmenu Help, Markdown Reference %}). A legtöbb esetben a MultiMarkdownt részesítjük előnyben, de ha már processzorspecifikus szintaxist használó fájljai vannak, vagy problémákba ütközik egy Markdown-fájl MultiMarkdown segítségével történő előállítása során, válassza ki a Commonmarkot alapértelmezett processzorként.
