[iThoughtsX][it] mind maps saved as `.itmz` packages can be previewed directly in Marked once **Markdown Preview** is enabled inside iThoughtsX.

## Enable Markdown export channel

In iThoughtsX preferences, under **Advanced**, set **Markdown Preview** (wording may read "Markdown" or "Markdown Preview" depending on version), then pick whether exported topics include:

- **All topics** &#8212; full map breadth, or
- **Visible topics** &#8212; only branches you have expanded in the canvas.

Marked reads the Markdown representation from the packaged map and renders it using your active style sheet.

Use **Outline mode** commands in Marked's {% appmenu Gear %} menu when you want APA- or Decimal-style outline numbering on the converted outline.

Save in iThoughtsX to push updates into Marked; the watcher notices filesystem changes automatically.
