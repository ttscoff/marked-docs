## Disabilita gli ID di intestazione automatici

Contrassegnato include un'opzione per disabilitare la generazione automatica dell'ID dell'intestazione. Puoi trovare questa opzione in {% prefspane Processor %}.

## ID delle note a piè di pagina casuali

Nel riquadro **Processore**, puoi selezionare "Utilizza ID note a piè di pagina casuali" per generare identificatori di note a piè di pagina casuali, che aiutano a evitare conflitti quando più documenti vengono visualizzati su una pagina web. Questa opzione è disponibile solo quando si utilizza il processore MultiMarkdown.

## Processo di markdown all'interno dell'HTML

Per impostazione predefinita, la sintassi Markdown all'interno dei tag del blocco HTML viene solitamente ignorata dai processori Markdown. Questa opzione forza Marked a continuare l'elaborazione all'interno degli elementi del blocco. Tieni presente che alcuni markup potrebbero causare problemi.
