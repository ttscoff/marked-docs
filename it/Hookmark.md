# <%= @title %>

I collegamenti [Hookmark](https://hookproductivity.com/) possono puntare a file e risorse a cui fa riferimento Markdown, comprese le immagini.

## Risoluzione URL hookmark

Marked può risolvere `hook://` collegamenti per immagini e altre risorse in un file quando l'integrazione Hookmark è abilitata.

## Abilita nelle Impostazioni

Apri {% prefspane Apps %} e attiva l'opzione relativa a Hookmark. Una volta abilitato, Marked risolve gli URL Hookmark durante il rendering in modo che le risorse collegate vengano visualizzate normalmente nell'anteprima e nell'esportazione.
