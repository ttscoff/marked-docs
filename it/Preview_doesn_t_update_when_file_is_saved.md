##La macchina del tempo

Sono stati segnalati alcuni problemi relativi ad aggiornamenti lenti (o inesistenti) durante l'esecuzione di Time Machine. Le funzionalità di aggiornamento di Marked potrebbero essere compromesse se Time Machine utilizza la parte dell'unità in cui si trova il file in quel momento. Questo di solito si risolve da solo quando Time Machine termina la sua corsa.

Se riscontri problemi con Marked che non risponde agli aggiornamenti dopo che Time Machine ha completato l'esecuzione o in un momento in cui Time Machine non è affatto in esecuzione, [invia un ticket](https://support.markedapp.com/questions/add) e risolveremo il problema insieme a te.
