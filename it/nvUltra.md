[nvUltra] [nvu] indicizza i taccuini di testo normale sul disco e può consegnare le note selezionate a Marked con una scorciatoia incorporata.

## Anteprima dei file in Contrassegnato

Nel **Navigatore cartelle** di nvUltra (o ovunque si sfoglino le note), **fare clic tenendo premuto il tasto Control** sui file per aprire il menu contestuale e scegliere **Anteprima file in Contrassegnati**.

![][nvultramenu]

Marked apre le normali anteprime supportate da file in modo che il tuo MultiMarkdown venga visualizzato con stili, segnalibri, attivatori della minimappa e opzioni di esportazione contrassegnati. Usalo quando vuoi passare direttamente da nvUltra senza trascinare i file sul Dock.

## Alternativa: controllo delle cartelle

Hai bisogno che Marked segua **qualunque nota tu abbia toccato più recentemente** in un'intera cartella? Rilascia invece la cartella del taccuino su Marked; vedere **[Controllo cartelle](Folder_Watching.html)** per il controllo in stile nvALT, lo scorrimento fino alle modifiche e la differenza tra il flusso di lavoro e l'apertura di file espliciti.
