[CriticMarkup](http://criticmarkup.com/) により、編集者は Markdown ドキュメント内でフィードバックを提供できます。 {% prefspane  Proofing %} で有効にすると、CriticMarkup 構文が検出されるたびにマーク付きプレビューに 3 つのタブが表示されます。挿入、削除、コメントをインラインで表示するマークアップ バージョン、編集前の元のバージョン、およびすべての編集が受け入れられた場合に表示される編集済みバージョンです。

歯車メニューから、または {% kbd ctrl shift C %} を使用して、ドキュメントの CriticMarkup サポートを切り替えることができます。

{% kbd shift cmd [ %} と {% kbd shift cmd ] %} を使用して、CriticMarkup プレビューのビューをすばやく切り替えます

CriticMarkup がアクティブな場合、単語数とドキュメントの統計は、現在いるビューに基づいて変更されます。これにより、ドキュメントの元のバージョンと編集されたバージョンの間で、基本的な統計と単語、文、および文字数の比較が可能になります。

CriticMarkup およびお気に入りのテキスト エディタとの統合の詳細については、[CriticMarkup ホームページ](http://criticmarkup.com/) を参照してください。
