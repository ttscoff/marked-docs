# <%= @title %>

[Curio][curio] は、編集中のアイテムを Marked の **ストリーミング プレビュー** チャンネルにストリーミングできる、ビジュアルなブレインストーミングとメモ取りのアプリです。

## マークを付けるために選択されたストリーム

1. [マーク] で {% appmenu File, New, Streaming Preview %} を選択すると、ストリーミング プレビュー ウィンドウの準備が整います。
2. Curio で、**表示 → 選択したストリームをマーク付きにします** をオンにします。

ダブルクリックして Curio 内の図、メモ、リスト、またはその他の項目を編集すると、そのマークダウンがストリーミング プレビューに流れ、書き込み中に更新されます。

他の統合アプリと同じ `mkStreamingPreview` クリップボード契約に従って、ほぼリアルタイムでマークされた更新。[ストリーミング プレビュー](Streaming_Preview.html) の [技術詳細](Streaming_Preview.html#developers) を参照してください。

プレビューの更新が停止した場合は、[マーク] でストリーミング プレビュー ウィンドウが開いていることを確認し、Curio で **[選択されたストリームをマーク]** に一度切り替えます。
