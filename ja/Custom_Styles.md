```css
@media print {
  #wrapper p,
  body p,
  p {
    font-size: 9pt !important;
    line-height: 1.4 !important;
  }

  h1 {
    font-size: 16pt !important;
  }
}

.mkprinting body p,
.mkprinting p {

  line-height: 1.4 !important;
}

.mkprinting h1 {
  font-size: 16pt !important;
}
```
