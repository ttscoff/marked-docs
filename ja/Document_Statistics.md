## ワード数とドキュメントの統計

![][1]

   [1]: 画像/WordCount.jpg @2x width=840px height=61px class=center

単語数は下部のステータス バーに表示され、ステータス バーの下の {% prefspane General %} から有効または無効にできます。歯車メニューから単語数をクリックするか、Option-Command-S ({% kbd opt cmd S %}) を入力すると、プレビューまたはソース ウィンドウで詳細な統計を表示できます。オプション ({% kbd opt  %}) キーを押しながらクリックすると可読性統計が表示され、オプション + コマンド ({% kbd opt cmd %}) を押して詳細統計パネルが開きます。

テキストが選択されている場合、単語数の表示と段落/文/文字のポップアップが選択内容のみの情報で更新されます。

## 選択範囲のワード数

![テキスト選択時の単語数ポップアップ][2]

   [2]:images/wordcountforselection.jpg @2x width=749px height=144px class=center

プレビューでテキストを選択すると、ウィンドウの下部にある単語数に、選択したテキストのみの統計が表示されます。

{% prefspane Preview %} で「選択範囲の単語数を表示」が有効になっている場合、カーソルの横に小さなポップアップが表示され、選択したテキストの単語/行/文字数が表示されます。これは、マウスをポップアップから遠ざけるだけで閉じられます。

ズーム機能は、大きなテキストの塊をすばやく選択してカウントを取得するのに便利です。 {% kbd z %} と入力してズームアウトし、選択します。

## 可読性の統計

![可読性統計バー][3]

   [3]: 画像/スクリーンショット/mainwindow-feature-stats- lower-crop.jpg @2x width=1089px height=111px class=center

Flesch/Kincaid および霧指数からの追加統計は、{% kbd opt shift cmd S %} で利用できます。

### 可読性に関する情報

**読みやすさ:** スコアが高いほど、読みやすい内容であることを示します。数字が小さいほど、文章が読みにくくなります。

**90.0--100.0**: 平均的な 11 歳の学生
**60.0--70.0**: 13 ～ 15 歳の学生
**0.0--30.0**: 大卒

**Flesch-Kincaid Grade Level:** このテキストを理解するために通常必要な教育年数。

**ガンニングフォグ指数:** は英語の文章の読みやすさを測定します。この指数は、テキストを初めて読んで理解するために必要な正式な教育の年数を推定します。霧指数 12 には、米国の高校 3 年生 (約 18 歳) の読解レベルが必要です。

## 高度な統計

![高度な統計ポップアップ][adv]

[adv]: images/screenshots/mainwindow-feature-stats-info-crop.jpg @2x width=800px height=800px class=center

歯車メニューから [詳細な統計] を選択する ---- または {% kbd cmd I %} --- を押すと、平均の単語と文の長さ、平均の単語の複雑さなど、より高度な文書統計を含むパネルがドロップダウンされます。

### 流動的な高度な統計

![フローティング情報ウィンドウ][フローティング]

[floating]: images/floatinginfo.jpg @2x width=999px height=522px class=center

{% kbd shift cmd I %} を押すと、すべての詳細な統計情報と可読性情報を含むフローティング パネルが開きます。このパネルは、エディターに切り替えたときに最前面に表示されたままになるため、プレビューが表示されているかどうかに関係なく、保存するたびに統計が更新されるのを確認できます。 `<` アイコンを押すと、関連付けられたマーク付きプレビューが最前面に戻ります。オプションを押したまま同じボタンをクリックすると、デフォルトのテキスト エディター ({% prefspane Apps %} で設定) で Markdown ファイルが開きます。

### 単語ターゲット

執筆中に単語数の具体的な目標がある場合は、文書の先頭に「target:」メタデータ キーを追加できます。マークすると進捗状況が追跡され、詳細統計パネル ({% kbd cmd I %}) とフローティング統計 ({% kbd shift cmd I %}) に完了インジケーターが表示されます。

## 単語の繰り返しを視覚化する

歯車メニューから [単語の繰り返しを視覚化] を選択すると (または {% kbd ctrl cmd W %} を押すと) 特別なビューに切り替わり、テキスト以外の要素が削除され、文書内で繰り返されている単語が強調表示されます。繰り返される単語は明るいピンク色で強調表示され、強調表示された単語の上にマウスを置くと、文書全体で一致する単語が明るくなります。強調表示された単語をクリックすると、背景が暗くなり、さらに確認できるように強調表示が「固定」されます。

![単語の繰り返し][4]

[4]: images/screenshots/mainwindow-word-repetition.jpg @2x width=1369px height=1084px

このモードでは「ズーム」機能 (「z」と入力) も使用でき、繰り返される単語を強調表示し、文書全体をすばやくスキャンして、繰り返しが集中している場所を確認できます。

単語は「語幹」によって比較されます。つまり、「部分」、「一部」、および「一部」は繰り返し単語とみなされます。これにより、繰り返しの密度をチェックするときに音節と活用が考慮されます。

**範囲**

繰り返しチェックの範囲はドキュメントの下部ツールバーで変更でき、ドキュメントまたは段落に設定できます。ドキュメント モードがデフォルトです。 「段落」を選択すると、テキストの各ブロック内の繰り返しのみがカウントされます。繰り返しは文書全体で引き続き強調表示されますが、単一の段落内で単語が複数回出現する場合にのみカウントされます。

**言葉を無視する**

強調表示された単語を Option キーを押しながらクリックすると、単語とそのさまざまな活用形や複数形をすべて一時的に除外できます。一時的に無視された単語はプレビューの右下隅に表示されます。無視された単語リスト内の単語をクリックすると、その単語が視覚化に復元されます。

単語を永久に無視するには、[繰り返しを無視] タブの {% prefspane Proofing %} でリストを編集できます。このリストに 1 行に 1 つずつ追加される単語 (または単語の語幹) は常に無視されます。このリストに単語を自動的に追加するには、単語の繰り返しの視覚化で Option キーを押しながら Shift キーを押しながらクリックします。

**単語反復モードを終了します**

下部ツールバーの範囲セレクターの横にある「閉じる」アイコンを使用するか、Esc キーを押して、単語の繰り返しビューを閉じることができます。
