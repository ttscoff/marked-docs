## キーワードの強調表示

「マーク付き」でのキーワードの強調表示により、避けたい一般的なフレーズを見つけたり、代替用語を見つけたり、一般的な目的で強調表示したりすることができます。各カテゴリに一致するキーワードのリストは、{% prefspane Proofing %} で編集できます。

歯車メニュー ({% appmenu {{gear}}, Highlight Keywords %}) から {% kbd shift cmd H %} で強調表示を有効にするか、左下 (歯車メニューの近く) にある強調表示アイコンを使用してキーワード ドロワーを開きます。引き出しはキーボード ショートカット {% kbd shift cmd K %} でも開くことができます。ハイライト表示は引き出しを開けると自動的に有効になり、引き出しの左側にあるスイッチでオン/オフを切り替えることができます。

## キーワード引き出し

![キーワード引き出し][kwd]

[kwd]: images/screenshots/mainwindow-feature-keywords-crop.jpg @2x width=1089px height=503px class=center

ハイライトを有効にすると表示されるキーワード ドロワーを使用すると、個々のハイライト タイプを有効または無効にする機能など、ハイライト オプションにすばやくアクセスできます。左側にある色付きラベルの縦列は、テキスト内のハイライトに対応しています。ラベルをクリックすると、そのキーワード タイプのハイライトが切り替わります。

各ラベルの左側にはターゲット アイコンがあります。これをクリックすると、ドキュメントの順序でハイライトの次のインスタンスまでドキュメントのスクロールが開始されます。ラベルの右側には、そのタイプのハイライトの合計数を示すカウントが表示されます。

キーボードを使用すると、ハイライト内をすばやく移動できます。 `[` と `]` を入力すると、最初はすべてのハイライトを前後に移動します。ターゲット アイコンをクリックすると、`[` と `]` がそのタイプのハイライトのみのナビゲーションに切り替わります。 `{` (Shift-[) および `}` (Shift-]) は、最後にクリックされたターゲットに関係なく、常にすべてのハイライトに移動します。

強調表示された単語または語句をクリックすると、そのタイプがナビゲーションのターゲットになり、`[` または `]` を使用すると、文書内のその位置から移動します。

## キーワードの編集

![校正設定][proofprefs]

  [proofprefs]:images/screenshots/preferences-Proofing.jpg @2x width=689px height=1031px class=preferencepane

デフォルトでは、Marked は [Plain English Campaign](http://www.plainenglish.co.uk) の使いすぎた単語やフレーズのリストを使用します。 {% prefspane Proofing %} で編集することで、これらを簡単に追加または置き換えることができます。各フィールドは自由形式のテキストであり、各行は検索フレーズとして解釈されます。先行するテキストと一致するにはフレーズまたは単語の先頭で `*` を使用し、ワイルドカードとして単一の文字と一致するには `?` を使用します。

正規表現は、式をスラッシュで囲んで使用できます。

    /\\S*?ly/

上記は、強調表示のために「ly」で終わる単語と一致します。 Marked のキーワード ハイライトの正規表現の構文は [JavaScript と同じ](http://www.regular-expressions.info/javascript.html) です。

## 一時的なキーワード

メモ帳を編集して、キーワード ドロワーに一時的なキーワードを追加することもできます。 {% prefspane Proofing %} フィールドと同様に、1 行に 1 つのキーワードまたは語句を追加します。正規表現を使用できます (スラッシュで囲みます)。一時キーワードを編集した後は、必ず「更新」ボタンをクリックして (または {% kbd cmd return  %} を押して) 変更を保存し、文書内で強調表示されていることを確認してください。

正規表現は一時キーワード フィールドでも機能し、テキストをスラッシュ (`/my expression\b/`) で囲むだけです。

一時キーワードは、キーワードの密度が重要な状況を対象としており、対象の単語が何回使用されたかをすぐに確認し、文書内のその位置を強調表示することができます。一時キーワードの一致数がドロワーに目立つように表示されます。

テキスト内で頻繁に使用されている単語を見つけるには、["単語の繰り返しの視覚化"][wordrep] コマンドも参照してください。

[wordrep]: Document_Statistics.html#visualizewordrepetition

## 受動態

マークは英語の文章における「受動態」の使用法を指摘します。 [ウィキペディアによる定義][受動的]として:

> 文法上の主語は、主動詞の主題または患者、つまり、動作を受ける、またはその状態が変化する人または物を表します。

[言語学者ジェフリー K. プルムによる投稿][gkp] について読むことができるように、受動態は悪ではありません。マークは受動態を使用して文章を指摘しますが、その妥当性についての判断はあなたにあります。

[passive]: https://en.wikipedia.org/wiki/Passive_voice
[gkp]: http://languagelog.ldc.upenn.edu/nll/?cat=54

## 単語が 2 倍になる

キーワードの強調表示が有効になっている場合、ダブルワード (例: 「the the」) がオレンジ色で自動的に強調表示されます。これは現在設定できませんが、校正には便利です。
