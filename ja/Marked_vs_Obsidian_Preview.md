# <%= @title %>

Obsidian には、Markdown プレビューが組み込まれています。 Marked 3 は、Obsidian や他の多くのエディターと連携する専用のプレビューおよび公開アプリです。このページは、各ツールがいつ適切に適合するかを判断するのに役立ちます。

## Obsidian の組み込みプレビューで十分な場合

* Obsidian の内側にのみ書き込みます
* 基本的なプレビューとプラグインはニーズを満たします
* スタンドアロンのエクスポートまたは校正ワークフローは必要ありません
* 最小限のセットアップですべてを 1 つのアプリにまとめたい

シンプルな単一アプリのソリューションだけで十分な場合には、Obsidian (および **Typora** などのアプリ) が最適です。

## マークされている場合は、より良い選択です

* Obsidian だけでなく、**任意の** エディタからライブ プレビューが必要な場合
* カスタム CSS を使用したプロフェッショナルなエクスポート (PDF、DOCX、EPUB) が必要です
* 高度な校正、文書統計、キーワードの強調表示が必要な場合
* Markdown と並行して Scrivener、Word、または MarsEdit を使用して作業する場合
* [カスタム スタイル] で公開品質のテーマが必要です(Custom_Styles.html)

Marked のプレビューには、Obsidian、Typora、および同様のアプリの組み込みプレビューが単独では提供しない機能や、プラグインや回避策を必要とする多くの機能が含まれています。

## 並べて比較

|特集 |マーク 3 |黒曜石のプレビュー |
| :-- | :-- | :-- |
|任意の Mac エディタからのライブ プレビュー |はい |黒曜石ファイルのみ |
|専用プレビュー+エクスポートアプリ |はい |メモアプリ |
|カスタム CSS テーマ |はい |テーマ/プラグイン |
|校正ツール |上級 |基本 |
| PDF、DOCX、EPUB エクスポート |はい |プラグイン |
| Scrivener / Word 統合 |はい |いいえ |

Typora と Marked 2 を含むより広範な比較については、[マーケティング比較ページ](https://markedapp.com/markdown-preview/) を参照してください。

## 両方を使用: Obsidian + Marked ワークフロー

Marked の恩恵を受けるために、Obsidian のプレビューを置き換えたり、Typora の単一ウィンドウ編集を放棄したりする必要はありません。多くのライターは、好みのエディターをそのまま使用して、他のアプリが提供していないプレビュー機能やエクスポート オプションにマークを追加しています。

1. いつものように Obsidian (または Typora、VS Code など) で書き込みます。
2. Vault フォルダーを Marked にドラッグするか、[Marked 3 Obsidian プラグイン](Obsidian.html) を使用します。
3. Marked は Vault を監視し、最後に編集されたメモをプレビューします。
4. ドラフトの準備ができたら、校正、文書統計、カスタム テーマ、およびエクスポートに「マーク付き」を使用します。 「エクスポート」(Exporting.html)をご覧ください。
