# <%= @title %>

[MindNode][mn] ドキュメントを Marked にドラッグします (または、MindNode のバージョンに応じて、MindNode の **Open in Marked** / プレビュー コマンドを使用します)。 Marked はファイルを MindNode の Markdown エクスポートのように扱うため、マップの階層がプレビューの見出しとリストになります。

## ストリーミング プレビュー (MindNode 2026.3+)

I> **MindNode 2026.3** 以降では、マークされた [ストリーミング プレビュー](Streaming_Preview.html) が追加されます。 MindNode で、**表示 → マーク付きプレビュー** を選択して開きます。マップを変更すると、プレビューがライブで更新されます。

## MindNode Classic と MindNode 7+

W> 現在の統合は **MindNode Classic** バンドルをターゲットとしています。新しい MindNode リリースでは、異なるパッケージ形式が使用されます。マップを開くことが失敗するか、マークに **「MindNode から開いてください」** シートが表示される場合は、プレビュー用に準備されていない未加工の `.mindnode` パッケージをドラッグするのではなく、MindNode の **詳細** メニュー (通常は **別のアプリケーションで開く**/マーク付き) からマップを準備するか、互換性のあるバンドルをエクスポートします。

ファイルは Marked が読み取る前に MindNode によって準備されるため、Marked での更新を期待する前に必ず MindNode に保存してください。
