OPML アウトライン [OmniOutliner][oo] エクスポートは、Markdown と同じように Marked にドラッグできます。 Marked は、OmniOutliner (または汎用 OPML) 構造に基づいて、アウトラインを読み取り可能な Markdown 見出しとネストされたリストに変換します。

{% prefspane Preview %} では、**OPML アウトラインの埋め込み** と **OmniOutliner (.ooutline) パッケージ** を **マーメイド マインド マップ** として個別のスイッチで制御します。ダイアグラム ビューを使用する場合は、直接ネストされた Markdown リストを使用します。

OmniOutliner とは別に、Fargo、OutlineEdit、iThoughts (OPML エクスポート)、またはマインドマップ アプリなどのツールからの汎用 `.opml` ファイルは同じように動作します。
