## タイムマシン

Time Machine の実行中に更新が遅い (または存在しない) という問題がいくつか報告されています。 Time Machine がその時点でファイルが存在するドライブの部分を使用している場合、Marked の更新機能が損なわれる可能性があります。これは通常、Time Machine の実行が終了すると自動的に修復されます。

Time Machine の実行が完了した後、または Time Machine がまったく実行されていないときに Marked がアップデートに応答しないという問題が発生した場合は、[チケットを提出](https://support.markedapp.com/questions/add)してください。問題解決をお手伝いします。
