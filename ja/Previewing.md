## スタイルの選択

![][1]

   [1]: 画像/StylePicker.jpg @2x width=896px height=195px class=center

{% prefspane Style %} で新しいドキュメントのデフォルトのスタイルを設定できます。ウィンドウ設定でツールバーのスタイル メニューを有効にしている場合は、プレビュー ウィンドウから直接ドキュメントごとにスタイルを調整できます。選択したスタイルは記憶され、エクスポートおよび印刷オプションの最初の選択肢になります。

スタイル設定で追加されたカスタム スタイルは、両方のメニューで使用できるようになります。

スタイルはキーボード ショートカットで選択できます。スタイル メニューを開くと、各スタイルのキーボード ショートカットが表示されます。キーボード ショートカットはスタイルの順序で割り当てられます。リストの最初の 9 スタイルには {% kbd cmd 1 %} -- {% kbd cmd 9 %} でアクセスでき、次の 10 スタイルには {% kbd cmd opt 1 %} -- {% kbd cmd opt 0 %} などでアクセスできます。

## アウトラインモード

ドキュメントが階層リスト (マインド マップやアウトライン アプリケーションから生成されたものなど) である場合は、[歯車] メニューからアウトライン モードを有効にして、APA または Decimal アウトライン スタイルで特別な書式設定を適用できます。

{% prefspane Style %} では、特定のファイル拡張子に対して自動アウトライン モードを有効にすることができます。

## テキストズーム

![][2]

   [2]:images/textzoom.jpg @2x width=800px height=414px class=center

{% kbd cmd shift + %} と {% kbd cmd shift - %} を使用してテキスト サイズを変更するか、メニューバーのプレビューの下にあるズーム メニューまたはドキュメント ウィンドウの歯車メニューを使用できます。 Marked は、次回 (および毎回) 行う変更を記憶します。 {% kbd cmd 0 %} を使用してズームを 100% にリセットします (または、ズーム メニューから **ズーム リセット** にアクセスします)。

## ダークモード/ハイコントラスト

暗い背景に明るいテキストを使いたい場合は、Marked が対応します。 __Preview__ メニューでは、{% appmenu Preview, Dark Mode ({{opt}}{{cmd}}I) %} デフォルト スキームの色を反転して、明暗の結果を作成できます。また、カスタム テーマが [適切に構築されている] (Writing_Custom_CSS.html) 場合は、そこでも同様に機能します。

## ステータスバーの表示/非表示

プレビュー ウィンドウの下部にあるステータス バーは、{% appmenu Preview, Show Status Bar ({{ctrl}}/) %} メニュー項目で切り替えることができます。非表示になっている場合は、プレビューの下部にあるスペースにマウスを移動することで表示および操作できます。
