### プロファイルのエクスポート

プロファイルのエクスポート
: ポップアップ メニューから保存されているプロファイルを選択します。プロファイルには、ワークフロー (印刷 PDF と Web HTML など) をすばやく切り替えるためのエクスポート設定の完全なセットが保存されます。 「プロファイルのエクスポート」(Exporting.html#export-profiles)を参照してください。

新しい
: 現在の設定から新しいプロファイルを作成します。

管理する
: プロファイル マネージャーを開いて、プロファイルの名前変更、複製、または削除を行います。

### 印刷/PDF

PDF のエクスポートまたは印刷時にリンク/ハイライトを無効にする
: 印刷時にリンクから書式設定 (下線と色) を削除します。

URLをテキスト注釈として含める
: PDF を印刷またはエクスポートすると、リンクの URL がリンクされたテキストの後に表示されます。

横罫線を改ページに置き換える
: 文書内の水平罫線を改ページに変更します (これにより、脚注も新しいページに強制的に追加されます)。

HTMLをコピーするときに画像を埋め込む
: 有効にすると、HTML をクリップボードにコピーすると、ローカル画像がスキャンされ、Base64 エンコードされてソース コードにデータ URL として含められます。

背景色と画像を印刷する
: デフォルトでは、マークは白い背景で印刷/エクスポートされます。カスタム テーマの背景色と画像を含める場合は、このオプションを有効にします。

孤立した見出しを防ぐ
: このオプションを使用すると、次のセクションの見出しが、後続のコンテンツなしでページの下部に表示されなくなります。

最初の H1 を除外
: H1 を改ページとして使用する場合、文書内の最初のレベル 1 の見出しを無視します。

最初の H1 をフォールバック タイトルとして使用する
: Bear やストリーミング プレビューなどのアプリを使用する場合、このオプションを使用すると、ドキュメント内の最初の H1 の内容でファイル名をオーバーライドできます。 「タイトル」のメタデータが指定されている場合は、それが常に優先されます。

前に改ページを追加する
: レベル 1/2 ヘッダーをセクション区切りとして使用し、常に新しいページで開始します。文書内の脚注の前に常に改ページを追加するには、「脚注」を選択します。

プレビューで改ページを指定する
: <!\--BREAK\--> 構文またはエクスポート設定の自動ブレーク オプションをオンにしてブレークが挿入された場所に薄い破線を表示します。

エクスポート/印刷用のカスタムフォントサイズ
: 設定すると、選択したポイント設定に基づいてすべてのテキストが拡大縮小されます (デフォルトは 10 ポイントを基準にします)。

余白
: 上下左右の印刷マージン (ポイント単位で指定) を定義します。

目次を印刷する
: 印刷物/PDF に自動目次を含めます。

改ページ後の
: 目次を挿入した後、自動的に新しいページに移動します。

目次レベルマーカー
: ドロップダウンを使用して、目次のインデントのレベルごとにリスト項目マーカーを設定します。

### ヘッダーとフッター

フォント、ロゴ、ヘッダー/フッター フィールド、日付と時刻の形式、最初のページの挿入、ページ番号のオフセット、枠線を構成します。フィールド プレースホルダーには、`%title`、`%logo`、`%page`、`%total`、`%date`、`%time`、`%h1`、`%h2` などが含まれます。

プレースホルダーのリファレンスと例については、[エクスポート](Exporting.html)の[ヘッダーとフッター](Exporting.html#headersandfooters)を参照してください。 `%date` および `%time` 形式コードについては、[日付と時刻の形式](Exporting.html#dateandtimeformats) を参照してください。

最初のページに含める
: ヘッダーおよび/またはフッターのオプションがチェックされていない場合、最初のページは指定されたタイプで印刷されません。

日付形式
: ヘッダーとフッターの `%date` の strftime スタイルのフォーマット文字列。 [日付と時刻の形式](Exporting.html#dateandtimeformats)を参照してください。

時刻形式
: ヘッダーとフッターの `%time` の strftime スタイルのフォーマット文字列。 [日付と時刻の形式](Exporting.html#dateandtimeformats)を参照してください。

ページ番号のオフセット
: ページ番号が何番目から始まるかを調整するために使用します。たとえば、最初のページに目次があり、番号付けを 2 ページ目から開始したい場合は、オフセットを -1 に設定します。ページ 2 がページ 1 になり、それに応じて合計ページが調整されます。

境界線
: ヘッダーの下および/またはフッターの上に水平線を印刷します。

デフォルトのフォーマットを復元する
: 日付と時刻の形式文字列を Marked のデフォルト (`%m-%d-%Y` および `%I:%M %p`) にリセットします。 [日付と時刻の形式](Exporting.html#dateandtimeformats)を参照してください。
