### レイアウトとタイポグラフィー

プレビューでのテキスト幅を制限する
: スライダーを使用して、プレビュー本文の最大幅を設定します (ピクセル単位)。

段落内の自動ハイフネーション
: 単語を自動的にハイフンで区切ることを許可します。

見出しと段落の未亡人を防ぐ
: 単一の単語が新しい行に折り返されるのを防ぐために、見出しと段落の最後の 2 つの単語の間に非改行スペースを強制します。

活字的に正しい引用符と句読点を生成する
: スマート クオート、省略記号変換、その他のタイポグラフィ機能 (MultiMarkdown) には SmartyPants を使用します。

脚注マーカーを角括弧で囲みます
: チェックすると、脚注マーカーにデフォルトの MultiMarkdown 書式設定が使用されます ([1])。角括弧を削除するにはチェックを外します。

拡張機能のアウトラインを有効にする
: リストされた拡張子を持つファイルのアウトライン モードを自動的にオンにします。

APA スタイルを使用する
: デフォルトの 10 進形式の代わりに APA スタイルのアウトラインを使用します。

逐語的 (コード) ブロックを詩としてスタイル設定する
: チェックすると、タブ インデント、フェンスで囲まれたコード、またはインクルードされたコードがコード ブロックではなく詩として表示されます (構文の強調表示やテーマに応じた特別なスタイルはありません)。

テーマがコードブロック内でテキストを折り返すことを許可する
: チェックすると、テーマで `pre>code` ブロック内での折り返しが許可されます。チェックを外した場合、水平オーバーフローは常にスクロールします。

コードを常にラップする
: テーマの設定に関係なく、コード ブロックを強制的にラップします (テーマのラップ動作をオーバーライドします)。

RTL テキストの検出とスタイル設定
: ドキュメント内の要素ごとに言語を検出し、それに応じて右から左にスタイルを設定します。

### テーマ

スタイルの管理
：[スタイルマネージャー](Style_Manager.html)ウィンドウを開きます。ドライブから CSS ファイルを追加すると、スタイル ピッカー メニューに表示されます。 `Add New Style` ボタンを使用するか、CSS ファイルをこのウィンドウにドラッグします。ドラッグして順序を変更し、チェックボックスを使用してスタイルを有効または無効にします。

その他のテーマ
: オンライン テーマ ギャラリーを開いて、追加のスタイルを参照してインストールします。

デフォルトのスタイル
: ここで選択したスタイルは、[ドキュメント固有のスタイルがメタデータで示されている](Per-Document_Settings.html) (例: "マークされたスタイル: Grump") を除き、すべての新しいウィンドウにロードされます。

CSS の変更を追跡する
: これを有効にすると、Marked は現在のスタイルのディスク変更を監視し、カスタム スタイルの編集や Web 開発を支援します。

追加のCSS
: ここで追加した CSS は、すべてのテーマの通常のスタイルシートの後にインクルードされます。とりわけ、これを使用すると、内部スタイルを編集せずに設定全体をオーバーライドできます。
: これはすべてのドキュメントとすべてのスタイルに適用されます。条件に基づいてドキュメントにカスタム CSS を適用する場合は、{% prefspane Processor %} のカスタム ルールを使用します。

### スクリプトを含める

構文の強調表示
: コード ブロックに対して、highlight.js [構文の強調表示](Syntax_Highlighting.html) をオンにします。ドロップダウンからテーマを選択します。
: **[言語が指定されている場合のみ**] がチェックされている場合、構文の強調表示は、言語が指定されているフェンスで囲まれたコード ブロックにのみ適用されます。

MathJax を有効にする
: MathML 数式を表示するために [MathJax](MathJax.html) をロードします。ドロップダウンから **ローカル** (バンドル) または **CDN** を選択します。
: **追加パッケージ** は、追加の MathJax パッケージ (物理学や化学など) を含めるシートを開きます。
: **高度な構成** は、カスタム MathJax 構成のシートを開きます。

KaTeX を有効にする
: MathJax の代替として [KaTeX](Mathjax.html#katex) をロードします。どちらか一方のみを選択できます。

数の方程式
: 該当する場合、マークはレンダリングされた方程式に図番号を追加します。番号付けには **左側** または **右側** を選択します。 MathJax を使用している場合は、**AMS のみ** を選択して、AMS 方程式のみに番号を付けることができます。

人魚
: CDN から [mermaid.js](https://mermaid.js) をロードして、マークダウン スタイルの図作成を有効にします。ドキュメントが更新されるたびに人魚図をレンダリングするために必要なフックが自動的に組み込まれます。

図のパンとズーム
: 人魚図が存在する場合、{% kbd cmd %} スクロールでズームを有効にし、クリックしてドラッグしてパンします。
