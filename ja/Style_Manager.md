## スタイルの追加

新しいカスタム スタイルをスタイル マネージャーに追加するには、いくつかの方法があります。

### 追加ボタン

[**新しいスタイルの追加**] ボタンをクリックしてファイル ピッカーを開きます
ここで、インポートする 1 つ以上の CSS ファイルを選択できます。選択したファイルは、
スタイル マネージャーに追加され、デフォルトで有効になります。

### ドラッグアンドドロップ

CSS ファイルをスタイル マネージャー ウィンドウに直接ドラッグできます。ドラッグすると
ファイルをウィンドウ上に置くと、「カスタム スタイルの追加」を示すオーバーレイが表示されます (または
複数のファイルの場合は「N 個のカスタム スタイルを追加」)。ファイルをドロップしてインポートします。

CSS ファイルをテーブル内の特定の位置にドラッグすることもできます (ドロップ)。
インジケーターは新しいスタイルが挿入される場所を示し、制御できるようにします。
インポートと配置の両方を 1 つのアクションで実行できます。

CSS ファイルを {% prefspane Style %} 設定ペインにドラッグすると、
それらをインポートすると、スタイル マネージャーが自動的に開きます。
