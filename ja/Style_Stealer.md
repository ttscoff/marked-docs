# <%= @title %>

あらゆる Web サイトからスタイルを抽出して盗用します。

## スタイルスティーラーとは何ですか?

Style Stealer は、Web サイトから CSS スタイルを抽出し、[カスタム スタイル](Custom_Styles.html) として Markdown ドキュメントに適用できるツールです。以下のような方に最適です。

- お気に入りの Web サイトのスタイルに合わせたい **ブロガー**
- 特定のブランドまたは出版物に一致するドキュメントを作成する必要がある **ライター**
- 既存の設計システムを使用して迅速にプロトタイプを作成したい **開発者**
- **優れたデザインの Web サイトの外観と雰囲気をキャプチャしたいと考えている人**

> Style Stealer は、サイトが明確なマークアップ分割で比較的適切にレイアウトされていることに依存します。すべてのサイトで機能するわけではありませんが、ほとんどのサイトで適切に機能するはずです。

> 最良の結果を得るには、できるだけ多くのテキスト コンテンツを含むページを入力してください。たとえば、ブログからスタイルを抽出するには、メインのインデックス ページではなく、記事または投稿を直接開きます。

## スタイルスティーラーの使用方法

### ステップ 1: スタイル スティーラーを開く

**ヘルプ** → **スタイル スティーラー**からスタイル スティーラーにアクセスします。

### ステップ 2: URL を入力します

URL フィールドに、スタイルを抽出する Web サイトのアドレスを入力します。 Style Stealer は、公的にアクセス可能なあらゆる Web サイトで動作します。サイトがペイウォールの内側にある場合、コンテンツを抽出するにはログインが必要になる場合があります。

![スタイル スティーラー プレビュー][プレビュー]

  [プレビュー]:images/style-stealer-preview.jpg @2x width=800

### ステップ 3: ロードとナビゲート

[**抽出**] をクリックするか、{% kbd return  %} を押して Web サイトを読み込みます。ロードすると、次のことが可能になります。

- **Command+リンクをクリックしてサイト内を移動**
- **コンテンツ領域の上にマウスを移動**すると、強調表示されます。
- スタイルを抽出したいメイン コンテンツ領域を **クリック**

選択するメイン コンテンツ領域には、見出し、段落、リストなどのみを含める必要があります。メニュー、サイドバー、またはその他の無関係なコンテンツを含むコンテンツ領域は選択しないでください。多くの場合、見出しは通常の段落コンテンツとは別のコンテナーに入れられます。このような場合は、まず両方を含む最小のコンテナを選択してみてください。結果が不十分な場合は、[**抽出**] を再度クリックし、段落を保持するコンテナのみを再選択します。

### ステップ 4: スタイルの抽出

コンテンツ領域をクリックすると、その領域に適用されるスタイルが抽出されます。プレビューは、すべての一般的な HTML 要素と、抽出されたスタイルが要素にどのように適用されるかを示す汎用ページとともに再ロードされます。

その後、このカスタム スタイルをカスタム CSS フォルダーに保存して、任意のドキュメントで使用できます。 [**保存**] ボタンをクリックするか、{% kbd cmd S %} を押して保存します。スタイルの名前は、入力した URL のドメインに基づいて付けられます。

## 何が抽出されるか

Style Stealer は、以下を含む包括的なスタイルのセットをキャプチャします。

### タイポグラフィ

- **フォント ファミリー**とすべての見出しレベル (H1 ～ H6) のサイズ
- **段落**のスタイル（行の高さと間隔を含む）
- **テキストの色**と背景の色
- **フォントのウェイト**とスタイル (太字、斜体など)

### レイアウトと間隔

- すべての要素の **マージンとパディング**
- **ボーダー**のスタイルと色
- **背景色** ダークテーマの本文背景を含む

### インタラクティブな要素

- **リンク スタイル** (ホバーと訪問した状態を含む)
- **ボタン**とフォーム要素のスタイル設定
- **リスト** スタイル (箇条書き、数字、インデント)

### 特別な機能

- **最初の段落**のスタイル設定
- **ブロック引用**の書式設定
- **コード**と事前にフォーマットされたテキストのスタイル設定
- **テーブル**のスタイリング
- **カスタム フォント フェイス**と Web フォント

## 高度な機能

### メディアのブロック

スタイル スティーラーは、メディア コンテンツ (ビデオ、画像、オーディオ) を自動的にブロックして、クラッシュを防ぎ、テキストのスタイルに集中します。これにより、メディアの多い Web サイトでもスムーズな抽出プロセスが保証されます。

### 疑似セレクターのサポート

このツールは、次のような CSS 疑似セレクターをキャプチャします。

- `:hover` リンクとボタンの状態
- `:visited` リンク状態
- `:first-child` 段落スタイル
- `::first-letter` ドロップキャップ用

### スマートフィルタリング

Style Stealer は次のものをインテリジェントにフィルタリングします。

- デフォルトのブラウザスタイル
- 不要なベンダープレフィックス
- 矛盾するルールまたは冗長なルール
- テキストを判読できなくなるようなスタイル

### デバッグモード

Style Stealer でデバッグ モードを有効にして、抽出プロセスの詳細なログを確認します。これは、トラブルシューティングや、どのスタイルがキャプチャされているかを理解するのに役立ちます。

## 最良の結果を得るためのヒント

### 適切なコンテンツ領域を選択する

- ヘッダー、サイドバー、フッターではなく、ページの **メイン コンテンツ エリア**をクリックします。
- 記事のテキスト、ブログ投稿、またはメインコンテンツが含まれる領域を探します。
- 大量の JavaScript や動的コンテンツが含まれる領域を避ける

### ダークテーマの処理

Style Stealer はボディの背景色を自動的にキャプチャするため、ダークテーマのスタイルを抽出するのに最適です。プレビューには、抽出されたダーク スタイルでコンテンツがどのように見えるかが表示されます。

### フォントに関する考慮事項

- **Web フォント**がキャプチャされ、抽出されたスタイルに含まれます
  - リモート URL からロードされたフォント (Google Fonts など) は、その URL を維持します。データ URL からロードされたフォントは、生成されたスタイルシートに複製されます。
- **システム フォント**は、別のシステム上で正常にフォールバックします。
- **フォントの読み込み**はプレビューで少し時間がかかる場合があります

### スタイルをテストする

抽出したスタイルを保存した後:

1. テストドキュメントに適用します。
2. 実際のコンテンツでどのように見えるかを確認します
3. 次の方法で調整します。
   1. {% prefspane Style %} を開きます
   2. [カスタム スタイル] テーブルで新しいスタイルを選択します。
   3. 「表示」をクリックして、Finder にファイルを表示します。
   4. プレーン テキスト エディターでファイルを開き (TextEdit はプレーン テキスト モードで動作します)、必要に応じて調整します。

## トラブルシューティング

### ウェブサイトが読み込まれない

- URL が正しく、公開されているかどうかを確認してください。
- 一部のサイトでは自動アクセスをブロックする場合があります
- 同じサイトの別のページを試してみる

### スタイルが違うように見える

- 抽出されたスタイルは、選択した特定のコンテンツに基づいています
- 一部のサイトでは複雑な CSS が使用されており、完全には翻訳されない可能性があります
- 追加の CSS を使用するか、スタイルシートを編集して微調整を行います

### スタイルがありません

- サイドバーやヘッダーではなく、メインのコンテンツ領域を選択していることを確認してください
- 一部のスタイルは JavaScript 経由で適用される可能性があり、キャプチャされません
- デバッグ コンソールで詳細な抽出情報を確認してください。

## キーボード ショートカット

- {% kbd return  %} - 抽出用の URL をロードします
- {% kbd cmd S %} - 抽出したスタイルをカスタム スタイル CSS ファイルに保存します
- {% kbd cmd  %} クリック - プレビュー中にリンクを移動します

## カスタム スタイルとの統合

抽出されたスタイルはカスタム CSS フォルダーに保存され、次のようになります。

- **スタイル メニューを通じて任意のドキュメントに適用**
- 任意のテキスト エディタを使用して **修正**
- CSS ファイルをコピーして他のユーザーと **共有**
- **他のカスタム スタイルと組み合わせる**

Style Stealer を使用すると、インターネット上で最も優れたデザインの Web サイトからインスピレーションを得た美しいスタイルのライブラリを簡単に構築できます。
