# <%= @title %>

[Ulysses][ulysses] はシートごとにプレーンな Markdown ファイルを作成しないため、Marked でプレビューしたい場合は現在のテキストを**エクスポート**します。

## マークされたものへのクイック エクスポート

1. Ulysses で {% kbd cmd 6 %} を押します (または、ツールバーのエクスポート ボタンをクリックします)。
2. 形式ドロップダウンを **テキスト** に設定します。
3. 二次形式を **Markdown** に設定します (TextBundle に関する以下の注を参照)。
4. [**次の方法で開く**] の横の下向き矢印をクリックし、[**マーク付き**] を選択します。
5. Tab キーを押して、主アクションが「**次で開く**」と表示されたら確認します。

最初のセットアップ後、{% kbd cmd 6 %} に続いて **Return** を押すだけで、アクティブなシート (または選択したシート) のマーク付きプレビューを更新できることがよくあります。

## 画像とアセット

Marked は、Ulysses からエクスポートされた **TextBundle** または **TextPack** を読み取ることもできます。同じエクスポート シートを使用しますが、マークダウンと一緒に埋め込み画像やその他のリソースが必要な場合は、バンドル オプションを選択します。
