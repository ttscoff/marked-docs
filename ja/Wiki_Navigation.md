# <%= @title %>

Marked には、簡単な `[[wiki]]` リンクを使用して、関連するテキスト ファイル間をすばやく移動できる Wiki ナビゲーション システムが含まれています。このシステムはシームレスなナビゲーション用に設計されており、現在のウィンドウでリンクを開くように構成されている場合に最適に機能します。

## 最適な設定

Wiki リンクを使用するには、**Convert `[[wiki links]]`** in {% prefspane Preview %} を有効にし、Marked が一致するファイルを検索するときに使用するデフォルトの拡張子を設定します。

最適なエクスペリエンスを得るには、{% prefspane Apps %} で **「テキスト ファイルへのリンクは次の場所で開く必要があります:」** を *「現在のウィンドウ」* に設定します。これにより、ナビゲーションが自然に感じられ、履歴が保存されます。

{% prefspane Proofing %} で **マークダウン構文エラーの強調表示** が有効になっている場合、現在のディレクトリ内のファイル名と一致しない `[[wiki link]]` 構文は赤色で強調表示され、参照されるファイルが存在しないことを示します。これらのハイライトは、ファイルが追加されると更新されますが、それは現在のファイルが保存または再ロードされた後にのみ更新されます。強調表示された欠落ファイルのリンクをクリックすると、必要に応じてデフォルトの拡張子を追加して新しいファイルを作成することが提案されます。新しい空のファイルがマーク付きで開かれ、[新しいドキュメントの編集] が有効になっている場合は、エディタが新しいファイルで開きます。

## Wiki リンクの仕組み

- **Wiki リンク** は、`[[wiki link]]` の形式を使用します。
- Wiki リンクをクリックすると、Marked は現在のドキュメントと **同じディレクトリ**内で一致するファイルを検索します。
- リンクに拡張子付きの完全なファイル名 (例: `[[notes.txt]]`) を指定しない限り、ファイルには Marked の設定で指定された拡張子 (例: `.md`) が付いている必要があります。
- ファイル名とは異なるテキストをリンクに使用する場合は、リンク内のパイプ (`|`) の後にテキストを追加します。 `[[wiki linking|A note about linking]]`、これは `[A note about linking](wiki-link.md)` と表示されます。
- Wiki リンクが `#` で始まる場合、同じページのアンカー リンクとして表示されます。アンカー名は小文字にし、すべてのスペースをダッシュ​​に置き換えることによって正規化されます。ダッシュのないヘッダー ID を作成するプロセッサ (MultiMarkdown など) の場合、例: `## wiki links` は ID `wikilinks` を取得します。このナビゲーションは壊れる可能性があります。このような場合は、必要に応じてパイプとタイトルを付けて、正しいリンク ID を指定します。 `[[#wikilinks|#wiki links]]`。

### 一致するファイル名

`[[wiki link]]` のようなリンクを使用すると、Marked は次のいずれかの名前を持つファイルを検索します (デフォルトの拡張子が `.md` であると仮定します)。

- `wiki link.md`
- `WikiLink.md`
- `wiki-link.md`
- `Wiki-Link.md`
- `wiki_link.md`
- `Wiki_Link.md`
- `wikilink.md`
- `WikiLink.md`
- (およびスペース、ダッシュ、アンダースコア、InterCap のその他の組み合わせ)

**すべての一致では大文字と小文字が区別されず、区切り文字も柔軟に使用できます。**
リンクに拡張子 (例: `[[notes.txt]]`) を指定すると、Marked はその正確なファイルを検索します。

## バックリンク

テキスト ファイルが開かれ、Wiki ナビゲーションが有効になっている場合、ディレクトリ内の残りのファイルが現在のファイルまで `[[links]]` スキャンされます。これはバックグラウンドで行われ、バックリンクが見つかった場合は、開いているドキュメントがそのリストで更新されます。バックリンクを表示するには、コメントのサイドバーが開いている必要があります。閉じている場合は、歯車メニュー (**校正 -> コメントの表示**) を使用するか、{% kbd ^@c %} を押して開きます。


## ナビゲーション履歴

Marked は Wiki リンクを通じてナビゲーションを追跡し、Web ブラウザと同じようにファイル履歴を**前後に移動**できます。

- **戻る:** {% kbd @[ %}
- **転送:** {% kbd @] %}

**履歴** メニューを使用して、以前にアクセスしたファイルにジャンプすることもできます。
