# <%= @title %>

[概要をズーム] を使用すると、プレビュー テキストが縮小されるため、文書のより多くの部分を一度に確認できます。これは、[ミニマップ](Document_Navigation.html#minimap) を開かずに構造を特定したり、セクションをスキャンしたり、長さを判断したりするのに役立ちます。

## Zoom の使用の概要

プレビューで {% kbd z %} を押すとズームアウトします。もう一度{% kbd z %}を押すと、通常のサイズに戻ります。ズーム中に **Escape** を押すか、{% kbd z %} 先端の閉じるコントロールを使用してズームインし直すこともできます。

## 単語の繰り返しの視覚化

**単語の繰り返し** ({% kbd ctrl cmd w %}) が開いている場合、同じ {% kbd z %} ショートカットが適用されます。ズームを切り替えると、文書内で繰り返される単語が集まっている場所を確認するのに役立ちます。
