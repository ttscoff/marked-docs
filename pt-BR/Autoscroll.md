# <%= @title %>

A rolagem automática move lentamente a visualização para leitura ou revisão com as mãos livres. É separado do [Índice](Document_Navigation.html) e do [Mini Mapa](Document_Navigation.html#minimap); ele rola a visualização atual automaticamente.

## Iniciando e parando

Acione a rolagem automática pressionando {% kbd s %}. A visualização avança na velocidade atual.

Um indicador no canto inferior esquerdo mostra a velocidade. Ajuste-o com as teclas de seta **Para cima** e **Para baixo**; segure **Shift** com as setas para passos maiores.

Return ({% kbd return %}) pausa ou retoma a rolagem.

{% kbd shift s %} inverte a direção de rolagem enquanto a rolagem automática está ativa.

Pressione {% kbd s %} novamente ou Escape para sair do modo de rolagem automática.

Ao usar Marcado como um teleprompter, você pode adicionar pausas com script na fonte do Markdown — consulte [Pausas de rolagem automática](Special_Syntax.html#pauses) em [Sintaxe especial marcada](Special_Syntax.html).
