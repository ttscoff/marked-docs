## Desativar IDs de cabeçalho automáticos

Marcado inclui uma opção para desativar a geração automática de ID de cabeçalho. Você pode encontrar esta opção em {% prefspane Processor %}.

## IDs aleatórios de notas de rodapé

No painel **Processador**, você pode marcar "Usar IDs aleatórios de notas de rodapé" para gerar identificadores aleatórios de notas de rodapé, o que ajuda a evitar conflitos quando vários documentos são exibidos em uma página da web. Esta opção só está disponível ao usar o processador MultiMarkdown.

## Processar Markdown dentro do HTML

Por padrão, a sintaxe Markdown dentro das tags de bloco HTML geralmente é ignorada pelos processadores Markdown. Esta opção força Marked a continuar o processamento nos elementos do bloco. Observe que algumas marcações podem causar problemas.
