# <%= @title %>

[MultiMarkdown Composer][mmc] trata Marked como um mecanismo de visualização externo.

Abra as preferências de **Visualização** do Composer e escolha Marcado como o aplicativo de visualização ou pressione {% kbd opt cmd P %} **Abrir em Marcado** no documento ativo sempre que desejar o sistema de estilo do Marcado, as ferramentas de exportação ou as sobreposições de revisão na mesma fonte Markdown que o Composer está editando.

Marcado observa o caminho do arquivo do Composer (ou conteúdo trocado, dependendo da versão) e é atualizado sempre que o Composer salva.

[mmc]: https://multimarkdown.com/composer4/
