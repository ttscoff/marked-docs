# <%= @title %>

## Seleção do processador

[MultiMarkdown](http://fletcherpenney.net/multimarkdown/ "MultiMarkdown") (por Fletcher Penney) é uma extensão de [Markdown](http://daringfireball.net/projects/markdown/ "Daring Fireball: Markdown") (por John Gruber). Toda a formatação padrão do Markdown é observada e o MultiMarkdown fornece sintaxes adicionais para elementos como tabelas e notas de rodapé.

## Sintaxe

Há um guia de sintaxe Markdown no menu de ajuda, ou você pode consultar a [documentação original de John Gruber](http://daringfireball.net/projects/markdown/syntax).

As extensões MultiMarkdown são documentadas por Fletcher Penney em [este documento](https://github.com/fletcher/MultiMarkdown/blob/master/Documentation/MultiMarkdown%20User%27s%20Guide.md).

## Renderização alternativa

Você pode escolher entre MultiMarkdown, Kramdown, CommonaMark e Desconto no {% prefspane Processor %}. Cada um oferece recursos adicionais com uma sintaxe ligeiramente diferente (veja {% appmenu Help, Markdown Reference %} para mais detalhes). Na maioria dos casos, MultiMarkdown é o preferido, mas se você já tiver arquivos usando sintaxe específica do processador ou tiver problemas para renderizar um arquivo Markdown com MultiMarkdown, selecione Commonmark como seu processador padrão.
