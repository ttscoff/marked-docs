# <%= @title %>

[Ulysses][ulysses] não grava um arquivo Markdown simples para cada planilha, então você **exporta** o texto atual quando quiser que o Marked o visualize.

## Exportação rápida para marcado

1. Pressione {% kbd cmd 6 %} no Ulysses (ou clique no botão exportar da barra de ferramentas).
2. Defina o menu suspenso de formato como **Texto**.
3. Defina o formato secundário para **Markdown** (veja a nota abaixo sobre TextBundle).
4. Clique na seta para baixo ao lado de **Abrir em** e escolha **Marcado**.
5. Tab até que a ação principal seja **Abrir em** e confirme.

Após a primeira configuração, {% kbd cmd 6 %} seguido de **Return** geralmente é suficiente para atualizar a visualização marcada da planilha ativa (ou das planilhas que você selecionou).

## Imagens e ativos

Marcado também pode ler exportações de **TextBundle** ou **TextPack** do Ulysses. Use a mesma planilha de exportação, mas escolha uma opção de pacote quando precisar de imagens incorporadas ou outros recursos junto com o Markdown.

[ulysses]: https://ulysses.app/
