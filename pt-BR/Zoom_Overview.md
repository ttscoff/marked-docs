# <%= @title %>

A visão geral do zoom reduz o texto de visualização para que você possa ver mais do documento de uma só vez - útil para localizar a estrutura, procurar uma seção ou avaliar o comprimento sem abrir o [Minimapa](Document_Navigation.html#minimap).

## Usando a visão geral do zoom

Pressione {% kbd z %} na visualização para diminuir o zoom. Pressione {% kbd z %} novamente para retornar ao tamanho normal. Durante o zoom, você também pode pressionar **Escape** ou usar o controle de dispensa na ponta {% kbd z %} para aumentar o zoom novamente.

## Visualização de repetição de palavras

O mesmo atalho {% kbd z %} se aplica quando **Repetição de palavras** ({% kbd ctrl cmd w %}) está aberto: alternar o zoom ajuda a ver onde as palavras repetidas se agrupam no documento.
