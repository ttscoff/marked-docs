# <%= @title %>

Os arquivos [iA Writer][ia] são documentos Markdown comuns em disco. Arraste o arquivo ativo do **Finder** ou do **ícone de proxy da barra de título** para Marcado para abri-lo.

Como o caminho de visualização corresponde ao arquivo do editor, as atualizações ao vivo se comportam como qualquer outro documento Markdown assistido.

A Marked entende que a maioria **inclui** fluxos de trabalho que dependem de padrões de caminho reconhecíveis; marcadores avançados de estilo `/block` do iA Writer são interpretados quando correspondem à sintaxe Marcados já suportam documentos modulares.
